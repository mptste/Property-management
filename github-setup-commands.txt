First time setup commands
==============================
1. git init
2. git add .
3. git commit -m "Commit message"
4. git remote add origin https://github.com/USERNAME/repository-name.git
5. git remote set-url origin https://USERNAME:personal-token@github.com/username/repository-name.git
6. git push --set-upstream origin master

Commands for modifying
==============================
1. git pull
2. git add .
3. git commit -m "Commit message"
4. git push
